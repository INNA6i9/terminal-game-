How it works

Front-end sends requests to the server; server calls the AI provider using a secret key; server streams back events to the browser so UI is realtime and responsive.

Best practices

  • Do not put keys in client code
  • Validate prompts and user inputs server-side
  • Rate limit and add usage quotas
